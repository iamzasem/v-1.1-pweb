Sätze mit "je ... desto ..." (Sentences with "the ... the ...")

A guide to using the 'je ... desto ...' structure in German to show proportional relationships, with examples, exercises, and resources.

Understanding "je ... desto ..."

Definition

'Je ... desto ...' is a comparative structure meaning "the ... the ...", showing how one thing increases or changes with another.

Formation

'Je' + comparative + clause (verb at end), 'desto' + comparative + clause (verb second), e.g., 'Je mehr ich lerne, desto klüger werde ich.'

Note: 'Je' and 'desto' don’t decline.

Rules

- Use comparatives ('mehr, besser').
- Shows proportionality.
- Comma separates clauses.
- Flexible with adjectives, adverbs, quantities.

Uses

1. Cause and effect: 'Je mehr ich esse, desto dicker werde ich.'
2. General truths: 'Je älter man wird, desto weiser wird man.'
3. Daily life: 'Je länger ich warte, desto ungeduldiger bin ich.'

  5. ___ höher ich steige, ___ müder werde ich.

Answers:

  1. Je mehr ich lerne, desto klüger werde ich.
  2. Je länger ich warte, desto ungeduldiger bin ich.
  3. Je weniger er sagt, desto besser ist es.
  4. Je öfter ich übe, desto schneller kann ich es.

Tips to Master "je ... desto ..."

  • Comparatives: Always use 'mehr, weniger, -er' (e.g., 'schneller').
  • Order: 'Je' first (verb at end), 'desto' second (verb second).
  • Comma: Don’t forget it between clauses.
  • Practice: Start with 'mehr' examples, then try adjectives/adverbs.
