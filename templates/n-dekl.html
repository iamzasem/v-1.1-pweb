N-Deklination (N-Declension)

A guide to 'N-Deklination'—masculine nouns adding '-n' or '-en' in all cases except nominative singular—with examples, exercises, and resources.

Understanding N-Declension

Definition

'N-Deklination' applies to masculine "weak nouns" that add '-n' or '-en' in accusative, dative, genitive, and plural, but not nominative singular.

Formation

- Nominative singular: Base form.
- Other cases/plural: Add '-n' (if ends in '-e') or '-en' (if ends in consonant).
- Articles decline normally.

Examples: 'Mensch' → 'Menschen', 'Affe' → 'Affen'.

Rules

- Masculine only.
- '-n'/'-en' in all but nominative singular.
- Plural matches oblique cases.
- Adjectives decline normally.

Which Nouns?

1. Living beings: 'Mensch, Affe, Bär.'
2. Roles: 'Kollege, Kunde.'
3. Specific: 'Name, Glaube.'
4. Foreign: 'Student, Herr.'

Uses

1. Identification: 'Der Mensch denkt.'
2. Description: 'Ein netter Kollege.'
3. General: 'Die Affen klettern.'

Exercise 2: Add the Correct Ending

Choose the correct form.

  1. Der Mensch → ___ (dative singular)
  2. Ein Affe → ___ (accusative singular)
  3. Die Kollegen → ___ (genitive plural)
  4. Der Student → ___ (nominative plural)
  5. Ein Name → ___ (dative singular)

Answers:

  1. dem Menschen
  2. einen Affen
  3. der Kollegen
  4. die Studenten
  5. einem Namen

Exercise 5: Mixed Examples

Choose the correct form.

  1. ___ liest ein Buch. (Mensch, nominative singular)
  2. Ich sehe ___. (Affe, accusative singular)
  3. ___ arbeiten zusammen. (Kollege, nominative plural)
  4. Ich spreche mit ___. (Student, dative singular)
  5. Die Farbe ___ ist rot. (Bär, genitive singular)

Answers:

  1. Der Mensch liest ein Buch.
  2. Ich sehe den Affen.
  3. Die Kollegen arbeiten zusammen.
  4. Ich spreche mit dem Studenten.
  5. Die Farbe des Bären ist rot.

Tips to Master N-Declension

  • Spot the Pattern: Look for '-n'/'-en' in non-nominative singular forms.
  • Memorize Key Nouns: Start with 'Mensch, Affe, Kollege, Student'.
  • Check Gender: Only masculine nouns use this.
  • Practice Cases: Use sentences to test all forms.
