Passiv mit Modalverb (Passive Voice with Modal Verbs)

A guide to using passive voice with modal verbs in German, combining werden + Partizip II with modals like können, müssen, and sollen, with examples, exercises, and resources.

Understanding Passive with Modal Verbs

What Is Passiv mit Modalverb?

It combines passive voice (focus on action/object) with a modal verb (ability, necessity, etc.), e.g., Das Buch kann gelesen werden (The book can be read).

Structure and Formation

Formation

Subject + modal verb (conjugated) + werden (infinitive) + Partizip II, optionally von + dative.

ActivePassive
Ich kann das Buch lesen.Das Buch kann gelesen werden.
Sie müssen die Arbeit machen.Die Arbeit muss gemacht werden.

Tenses

Present: kann … werden; Past: konnte … werden; Perfect (rare): ist … worden können.

Uses of Passive with Modal Verbs

Possibility (können)

  1. Das Lied kann gesungen werden. (The song can be sung.)

Necessity (müssen)

  1. Die Arbeit muss erledigt werden. (The work must be done.)

Obligation (sollen)

  1. Die Regeln sollen befolgt werden. (The rules should be followed.)

Exercises to Practice Passive with Modal Verbs

Exercise 1: Choose the Correct Modal Passive Form

Select the correct modal verb for the sentence.

  1. Die Aufgabe ___ heute erledigt werden.
  2. Das Fenster ___ geöffnet werden.
  3. Die Regeln ___ befolgt werden.

Answers:

  1. Die Aufgabe muss heute erledigt werden.
  2. Das Fenster kann geöffnet werden.
  3. Die Regeln sollen befolgt werden.

Exercise 2: Rewrite Active to Passive with Modal Verb

Choose the correct passive form of the active sentence.

  1. Ich kann das Buch lesen.
  2. Sie müssen die Arbeit machen.
  3. Er soll die Tür schließen.

Answers:

  1. Das Buch kann gelesen werden.
  2. Die Arbeit muss gemacht werden.
  3. Die Tür soll geschlossen werden.

Exercise 3: Translate

Choose the correct translation with passive and modal verb.

  1. The car can be washed today.
  2. The homework must be done now.
  3. The rules should be followed by us.

Answers:

  1. Das Auto kann heute gewaschen werden.
  2. Die Hausaufgaben müssen jetzt gemacht werden.
  3. Die Regeln sollen von uns befolgt werden.

Exercise 4: Correct the Mistake

Choose the corrected sentence with the right passive form.

  1. Das Buch kann werden gelesen.
  2. Die Arbeit muss werden gemacht.
  3. Die Tür soll geschlossen wird.

Answers:

  1. Das Buch kann gelesen werden.
  2. Die Arbeit muss gemacht werden.
  3. Die Tür soll geschlossen werden.

Exercise 5: Mixed Examples

Choose the correct modal verb for the sentence.

  1. Das Fenster ___ heute geschlossen werden.
  2. Die Suppe ___ warm serviert werden.
  3. Die Regeln ___ von uns befolgt werden.

Answers:

  1. Das Fenster muss heute geschlossen werden.
  2. Die Suppe kann warm serviert werden.
  3. Die Regeln sollen von uns befolgt werden.

Tips to Master Passive with Modal Verbs

  • Order Matters: Modal verb second, werden + Partizip II last (e.g., muss gemacht werden).
  • Modal Meaning: Kann = possible, muss = necessary, soll = expected.
  • Practice Tenses: Start with present (kann … werden), then try past (konnte … werden).
  • Check Verbs: Only use transitive verbs like bauen, not schlafen.
